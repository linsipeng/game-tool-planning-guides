# Minecraft Image-Art Section Plan

Large image art is easier to build and verify when divided into numbered sections.

## Before collecting blocks

- Choose final width, height, and orientation
- Define section dimensions
- Record the palette count per section
- Mark transparent or intentionally empty cells
- Choose a consistent build direction

Process the source with [Minecraft Image Art](https://mcimgart.com), then save one reference image for the whole build and one cropped reference per section.

Finish and inspect each section before moving on. Section boundaries create useful checkpoints and limit how far a placement error can propagate.
