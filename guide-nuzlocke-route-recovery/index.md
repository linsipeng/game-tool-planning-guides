# Recovering a Nuzlocke Plan After an Unexpected Loss

An unexpected faint changes more than one team slot. It can remove a safe switch, break a planned resistance chain, and force an underleveled reserve into an important fight. The useful response is not to rebuild the old plan exactly, but to write a new risk map.

Start by listing the jobs the lost Pokémon performed: damage, status control, pivoting, immunity, or emergency sacrifice. Then identify which upcoming encounters depended on those jobs. A replacement does not need the same species or moves if it covers the same decisions.

Review available encounters before spending rare items. Prefer options that solve several nearby fights rather than one distant boss. Record a fallback for critical hits, missed moves, and an opposing setup turn. A plan with a defined retreat or sacrifice is safer than one that only works when every roll is favorable.

Use [Nuzlocke Guide](https://nuzlockeguide.com) to confirm encounter context and game-specific planning details, then update the run notes before continuing. The recovery plan should describe what the new team can do, not what the old team used to do.
