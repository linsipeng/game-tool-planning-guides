# HSR Warp Planning Terms

- **Pity:** Pull count accumulated toward the next high-rarity result.
- **Hard pity:** Maximum pull boundary for the relevant banner result.
- **Soft pity:** Range where the chance rises before hard pity.
- **Guarantee:** State ensuring the next five-star is the featured result.
- **50/50:** Featured-character outcome when guarantee is not active.
- **Reserve:** Currency intentionally excluded from current spending.
- **Spendable passes:** Available passes after subtracting reserve.
- **Stop rule:** Precommitted boundary that ends the session.
- **Early result:** Five-star obtained before the planned ceiling.
- **Ending state:** Remaining passes, pity, and guarantee after stopping.
