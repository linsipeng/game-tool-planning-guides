# Game Tool Planning Guides

An open, versioned collection of practical worksheets for game planning tasks. Every guide explains a repeatable workflow and points to a focused web tool that can perform the related calculation or check.

## Guide and tool directory

- [Nuzlocke Boss Preflight Worksheet](nuzlocke-boss-preflight/) — [Nuzlocke Guide](https://nuzlockeguide.com)
- [Sky Session Timing and Route Template](sky-session-timing/) — [Sky Clock Online](https://skyclockonline.com)
- [Repeatable Genshin Build Test Case](genshin-build-test-case/) — [Genshin Impact Damage Calculator](https://gidamagecalc.com)
- [Honkai: Star Rail Warp Stop-Rule Worksheet](hsr-warp-stop-rule/) — [HSR Warp Calculator](https://hsrwarpcalc.com)
- [Minecraft Enchanted-Book Combination Tree](minecraft-enchant-tree/) — [Minecraft Enchantment Order Calculator](https://enchantorder.com)
- [Minecraft Image Art Build Checklist](minecraft-image-build/) — [Minecraft Image Art](https://mcimgart.com)
- [Survivor.io Upgrade Scenario Test](survivorio-upgrade-test/) — [Survivor.io Calculator](https://survivoriocalc.com)
- [Path of Exile Map Regex Test Cases](poe-regex-tests/) — [PoE Map Regex](https://poemapregex.com)

The rendered library is available on [GitHub Pages](https://linsipeng.github.io/game-tool-planning-guides/). The Markdown source remains public so corrections and link changes can be audited over time.
