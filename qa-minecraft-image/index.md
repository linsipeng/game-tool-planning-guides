# Minecraft Image Conversion QA Checklist

Use this checklist before exporting a block-art plan.

## Source

- [ ] Source dimensions are known.
- [ ] Transparency is intentional.
- [ ] Aspect ratio is correct.
- [ ] Small protected details are identified.

## Conversion

- [ ] Output dimensions match the build area.
- [ ] Palette restrictions reflect the game mode.
- [ ] Dithering is compared on and off.
- [ ] Transparent pixels stay empty.

## Build output

- [ ] Preview silhouette matches the source.
- [ ] Material list contains only allowed blocks.
- [ ] Block counts are practical.
- [ ] Orientation and row order are documented.

Run a final preview and material export with [Minecraft Image Art](https://mcimgart.com).
