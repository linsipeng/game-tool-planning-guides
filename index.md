# Game Tool Planning Guides

This small library contains practical, reusable worksheets for planning game challenges, comparing builds, scheduling events, preparing Minecraft projects, and testing map filters.

Each page is designed to be useful on its own. Use the worksheet to plan the task, then open the associated tool when you are ready to calculate, check, or execute it:

- [Nuzlocke Boss Preflight Worksheet](nuzlocke-boss-preflight/) — party and encounter planning; tool: [Nuzlocke Guide](https://nuzlockeguide.com)
- [Sky Session Timing and Route Template](sky-session-timing/) — daily event and route scheduling; tool: [Sky Clock Online](https://skyclockonline.com)
- [Repeatable Genshin Build Test Case](genshin-build-test-case/) — controlled artifact and rotation comparisons; tool: [Genshin Impact Damage Calculator](https://gidamagecalc.com)
- [Honkai: Star Rail Warp Stop-Rule Worksheet](hsr-warp-stop-rule/) — pity, guarantee, and pull-budget planning; tool: [HSR Warp Calculator](https://hsrwarpcalc.com)
- [Minecraft Enchanted-Book Combination Tree](minecraft-enchant-tree/) — prior-work and anvil-order planning; tool: [Minecraft Enchantment Order Calculator](https://enchantorder.com)
- [Minecraft Image Art Build Checklist](minecraft-image-build/) — palette and placement preparation; tool: [Minecraft Image Art](https://mcimgart.com)
- [Survivor.io Upgrade Scenario Test](survivorio-upgrade-test/) — upgrade-cost and resource comparisons; tool: [Survivor.io Calculator](https://survivoriocalc.com)
- [Path of Exile Map Regex Test Cases](poe-regex-tests/) — map-mod filter validation; tool: [PoE Map Regex](https://poemapregex.com)

The source Markdown is maintained alongside LinkPilot's publication records so each resource and outbound reference can be audited and updated.
