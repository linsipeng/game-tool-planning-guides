# HSR Pull Stop-Rule Worked Example

## Scenario

A player has 126 passes, 34 character pity, no guarantee, and wants to reserve 70 passes for the next banner.

## Boundaries

| Item | Value |
| --- | ---: |
| Available passes | 126 |
| Protected reserve | 70 |
| Spendable now | 56 |
| Current pity | 34 |
| Maximum ending pity | 90 |

## Decision

Stop after 56 pulls even if the five-star has not appeared. A surprise early five-star also ends the session so the result and guarantee state can be recorded before any further decision.

## Verification

Enter the starting pity, guarantee state, available passes, and reserve in [HSR Warp Calculator](https://hsrwarpcalc.com). The calculated ceiling must never spend the protected 70-pass reserve.
