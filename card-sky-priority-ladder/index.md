# Sky Short-Session Priority Ladder

When a session is shorter than expected, a priority ladder is more useful than a detailed route that can no longer fit.

## Define four levels

1. Time-limited events that cannot be repeated later
2. Daily goals with meaningful rewards
3. Social or group activities with fixed coordination
4. Optional collection and exploration

Estimate a stop time for every level. If the session falls behind, remove the lowest remaining level instead of rushing every activity.

Confirm event and cycle timing with [Sky Clock Online](https://skyclockonline.com). Keep five minutes unassigned for loading, travel mistakes, or joining friends. The result is a route that degrades cleanly instead of failing all at once.
