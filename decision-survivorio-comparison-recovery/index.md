# Survivor.io Comparison Recovery Decision Tree

Use this tree when two upgrade options produce a suspicious comparison.

1. Do both options use the same stage and character?
   - No: align the scenario.
   - Yes: inspect prerequisites.
2. Are all prerequisite upgrade costs included?
   - No: add them to the option total.
   - Yes: compare remaining currency.
3. Is one figure lifetime spending and the other next-level cost?
   - Yes: replace them with the same cost basis.
   - No: inspect breakpoint effects.
4. Does a breakpoint activate only after the final level?
   - Yes: calculate the complete path.
   - No: compare incremental output directly.

Create two separate but otherwise identical scenarios in [Survivor.io Calculator](https://survivoriocalc.com).
