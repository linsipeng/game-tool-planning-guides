# Minecraft Image Conversion Terms

- **Source dimensions:** Pixel width and height of the imported image.
- **Output dimensions:** Block width and height of the build plan.
- **Alpha channel:** Transparency information in the source.
- **Palette:** Set of blocks permitted during colour matching.
- **Dithering:** Mixing nearby colours to simulate an unavailable shade.
- **Aspect ratio:** Width-to-height relationship preserved during scaling.
- **Protected detail:** Small feature that must remain recognizable.
- **Material list:** Required block types and quantities.
- **Transparent pixel:** Source pixel intended to produce no placed block.
- **Preview:** Rendered approximation of the final block layout.

Compare palette and dithering options in [Minecraft Image Art](https://mcimgart.com) before exporting.
