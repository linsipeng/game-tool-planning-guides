# Survivor.io Calculation Field Reference

| Field | Meaning | Common mistake |
| --- | --- | --- |
| Character | Character and relevant bonuses | Switching characters between cases |
| Equipment grade | Rarity and evolution state | Recording level but not grade |
| Base attack | Starting value before the tested change | Using the post-upgrade value twice |
| Multipliers | Permanent applicable bonuses | Mixing temporary battle effects |
| Output metric | Damage, DPS, hits, or time to defeat | Comparing different metrics |
| Resource cost | Scarce materials required | Ranking by damage alone |
| Breakpoint | Practical encounter threshold crossed | Assuming every percentage gain matters |

Build the common baseline in the [Survivor.io Calculator](https://survivoriocalc.com). Duplicate it, change one field, and save both the numerical and practical result.
