# Minecraft Enchantment-Cost Audit

Use this audit when an anvil operation costs more than expected or becomes too expensive.

1. Confirm the game edition and version.
2. Record every input item's prior-work count.
3. Distinguish a fresh book from a visually identical combined book.
4. Check incompatible enchantments and maximum levels.
5. Rebuild the combination tree from the original inputs.

The displayed enchantments do not reveal the complete cost history. Rename operations and earlier combinations can affect the plan, so do not substitute an item without updating its history.

Enter labeled inputs into the [Minecraft Enchantment Order Calculator](https://enchantorder.com). Compare the proposed order with the operations already completed. If the real state has diverged, create a new tree from the current inventory instead of forcing the old sequence to continue.
