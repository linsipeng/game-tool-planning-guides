# PoE Map Regex Terms

- **Modifier text:** Exact map property string tested by the pattern.
- **Alternative:** One of several phrases separated by regex OR logic.
- **False positive:** Safe text incorrectly matched.
- **False negative:** Dangerous text incorrectly missed.
- **Control line:** Known safe or dangerous fixture used for testing.
- **Semantic fragment:** Distinctive phrase representing a modifier without matching unrelated words.
- **Compression:** Shortening a pattern while preserving its result set.
- **Mandatory rule:** Modifier that must always be rejected.
- **Optional rule:** Preference that may be enabled separately.
- **Regression fixture:** Saved test lines rerun after every edit.

Test both reject and allow controls with [PoE Map Regex](https://poemapregex.com).
