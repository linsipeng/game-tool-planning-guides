# Nuzlocke Limited-Resource Ledger

A challenge run becomes easier to reason about when limited resources are treated as decisions rather than inventory.

## Track before using

- Rare healing and status items
- One-time TMs or tutors
- Encounter-changing held items
- Evolution items
- Sacrifice candidates and safe pivots

For each resource, write the next two fights where it could matter. Do not spend it merely because it improves the current team; compare that gain with the value of preserving it for a known risk.

Use [Nuzlocke Guide](https://nuzlockeguide.com) to check encounter context, then record the reason whenever a limited resource is committed. A short ledger makes later route changes more deliberate and prevents forgetting why an item was being saved.
