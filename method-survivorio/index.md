# Survivor.io Upgrade Comparison Methodology

## Scenario model

Each option is a complete path from the same starting character, stage, equipment, currency, and output metric.

## Evaluation order

1. Freeze the shared scenario.
2. Record current levels and currency.
3. Include all prerequisite costs.
4. Calculate each target output.
5. Report remaining currency.
6. Separate immediate stage value from future scaling.

## Assumptions

- Next-level, path, and lifetime costs are distinct.
- Breakpoint effects activate only at their required target.
- Options must use the same output metric.
- Saved currency can have decision value even without immediate output.

Build isolated paths in [Survivor.io Calculator](https://survivoriocalc.com).
