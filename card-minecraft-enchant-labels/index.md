# Minecraft Enchanted-Book Label Sheet

Draw every proposed merge using these IDs. After an anvil operation, create a new label for the result instead of continuing to call it by the old book name.

Test the labeled tree with the [Minecraft Enchantment Order Calculator](https://enchantorder.com). This prevents two books with identical visible enchantments but different prior-work histories from being treated as interchangeable.
