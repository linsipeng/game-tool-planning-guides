# Survivor.io Upgrade Comparison QA Checklist

Use this checklist before selecting an upgrade path.

## Scenario

- [ ] Character, stage, and equipment assumptions are fixed.
- [ ] Current levels are accurate.
- [ ] Available currency is recorded.
- [ ] The next objective is explicit.

## Costs

- [ ] All prerequisites are included.
- [ ] Next-level cost is not confused with lifetime cost.
- [ ] Remaining currency is shown.
- [ ] Refund or reset assumptions are documented.

## Results

- [ ] Both options use the same output metric.
- [ ] Breakpoint effects activate at the correct level.
- [ ] Incremental and major-tier paths are compared fairly.
- [ ] Future savings are separated from current-stage output.

Create isolated scenarios in [Survivor.io Calculator](https://survivoriocalc.com) for each option.
