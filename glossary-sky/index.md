# Sky Schedule Terms

- **Daily reset:** The moment daily activities and rewards refresh.
- **Event window:** Period during which a timed activity is available.
- **Start interval:** Time between repeated event starts.
- **Travel buffer:** Time reserved to reach the event location.
- **Claim buffer:** Time reserved to finish and collect a reward.
- **Late-arrival cutoff:** Latest useful arrival time.
- **Exact fit:** A plan with no delay tolerance.
- **Fallback route:** Shorter route used when the primary schedule slips.
- **Local timezone:** Timezone used to display event and reset times.
- **Reset-sensitive step:** An action that loses value when reset occurs.
