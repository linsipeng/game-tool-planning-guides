# HSR Warp Planning Field Reference

| Field | Meaning | Common mistake |
| --- | --- | --- |
| Banner type | Character, light cone, or standard | Combining separate pity systems |
| Current pity | Pulls since the last five-star | Counting since the featured result |
| Guarantee | Whether the next five-star is featured | Inferring it from pity alone |
| Passes | Ready-to-use special passes | Double-counting converted currency |
| Stellar Jade | Convertible currency available | Including a protected reserve |
| Pull ceiling | Maximum spend for this banner | Moving the ceiling during the session |
| Stop rule | Result that ends the session | Defining it after an early loss |

Model the saved state with the [HSR Warp Calculator](https://hsrwarpcalc.com). If history is incomplete, record a range and the missing assumption.
