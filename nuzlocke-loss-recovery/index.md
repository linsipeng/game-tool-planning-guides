# Nuzlocke Loss Recovery Checklist

Use this checklist immediately after an unexpected faint. The goal is to rebuild team functions and route safety before continuing.

## Lost role inventory

- [ ] Damage role identified
- [ ] Defensive switch or immunity identified
- [ ] Speed control, status, or setup role identified
- [ ] Held item returned or marked unavailable
- [ ] One-time move or resource loss recorded

## Upcoming risk review

| Encounter | Lost role needed | Replacement option | Backup action |
| --- | --- | --- | --- |
| Next major fight | | | |
| Next route | | | |
| Next capture opportunity | | | |

## Replacement test

- [ ] Replacement covers the required job, not merely the same type
- [ ] Leveling path does not expose another critical team member
- [ ] New safe switch sequence is written down
- [ ] Critical-hit and missed-move responses are defined
- [ ] Limited items are committed only after comparing later fights

Check game-specific encounter and ruleset context with [Nuzlocke Guide](https://nuzlockeguide.com), then update the run notes with the new plan.
