# Sky Session Timing and Route Template

This template helps plan a short Sky: Children of the Light session around one timed event without turning the session into a race.

Check the current and upcoming cycles on [Sky Clock Online](https://skyclockonline.com) before filling in the times.

## Fallback plan

If the event is missed or delayed:

1. Complete the nearby secondary goal.
2. Check whether the next cycle fits the remaining session.
3. Skip the event if reaching it would consume the whole session.

## Group message

Send both the event time and departure time:

> Meet at ___ by ___. We leave for ___ at ___. If we miss it, we will ___ nearby.

## Review

After the session, record actual travel time and the largest delay. Add a small buffer next time. The purpose is dependable coordination with room for exploration, not maximum theoretical efficiency.
