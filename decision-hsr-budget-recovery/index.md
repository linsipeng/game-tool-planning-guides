# HSR Warp Budget Recovery Decision Tree

Use this tree when a pull plan appears to spend too much or reports the wrong ceiling.

1. Was current pity entered before calculating?
   - No: add it and recalculate.
   - Yes: verify the banner type.
2. Is the next five-star guaranteed?
   - Yes: preserve that state until a five-star result.
   - No: model both featured and lost outcomes.
3. Is a reserve protected?
   - Yes: subtract it from spendable passes.
   - No: use the full available balance.
4. Does the stop rule occur before the theoretical ceiling?
   - Yes: the stop rule is the effective maximum.
   - No: keep the hard-pity ceiling.

Recreate the starting state in [HSR Warp Calculator](https://hsrwarpcalc.com) and record the state after every five-star result.
