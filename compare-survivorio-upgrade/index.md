# Survivor.io Upgrade Path Comparison

| Path | Immediate gain | Path cost | Remaining currency | Best use |
| --- | --- | --- | --- | --- |
| Small weapon levels | Predictable | Low | High | Near-term stage damage |
| Equipment breakpoint | Large at threshold | Medium-high | Medium | Multiplier activation |
| Character level path | Broad | Variable | Medium | Multi-system improvement |
| Save currency | None now | Zero | Highest | Upcoming major tier |
| Mixed upgrades | Balanced | Medium | Low-medium | Several weak slots |

## Decision rules

- Include prerequisites in path cost.
- Use the same stage and output metric.
- Report remaining currency.
- Separate current-stage value from future scaling.
