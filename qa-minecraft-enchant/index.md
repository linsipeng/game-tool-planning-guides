# Minecraft Enchantment Order QA Checklist

Use this checklist before beginning an expensive anvil sequence.

## Inventory

- [ ] Every item and book is listed separately.
- [ ] Existing enchantments are exact.
- [ ] Prior-work history is recorded for each item.
- [ ] Incompatible enchantments are identified.

## Sequence

- [ ] Both left-right orientations were considered.
- [ ] Low-prior-work books are paired before expensive items.
- [ ] At least one alternative tree was costed.
- [ ] The final operation stays within the legal limit.

## Execution

- [ ] The in-game item matches the saved step before each operation.
- [ ] Renaming cost is included if used.
- [ ] No creative-mode assumption is mixed into survival planning.
- [ ] Total level cost is archived.

Model the complete inventory in [Minecraft Enchant Order](https://enchantorder.com) before using the anvil.
